# План разработки: Чайная Линия (Tea Layer)

**URL:** `tea.lyukos.com`
**Статус:** Планирование
**Принцип:** Чистая эстетика. Отсутствие CTA.

## 1. Концепция
Это личное пространство. "Слой жизни", куда можно заглянуть.
Не продает встречи. Не приглашает. Просто существует.

## 2. Структура

### Блок 1: Flow (Состояние)
- **Визуал:** Медленное видео процесса заваривания.
- **Звук:** Эмбиент (по желанию).
- **Текст:** Отсутствует или 1-2 фразы о ритме.

### Блок 2: Artifacts (Предметы)
- **Контент:** Фотографии чайной утвари.
- **Описание:** История предмета. Без цены. Без кнопки "Купить".

### Блок 3: Thoughts (Мысли)
- **Контент:** Короткие заметки о состоянии ума.

## 3. Навигация
- Нет меню.
- Нет кнопок "Связаться".
- Только ссылка "Назад в город" (на lyukos.com).
