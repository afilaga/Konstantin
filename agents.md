# Инструкции для Агентов (Agents Instructions)

Этот файл определяет правила работы и приоритеты для всех AI-агентов, работающих над проектом **Konstantin**.

## 1. Глобальный Приоритет: Скелет и Структура
- **Фокус:** Мы строим фундамент и структуру проекта ("Скелет").
- **Игнорировать:** На текущем этапе НЕ фокусируемся на маркетинговых метриках (ROI, конверсия, воронки продаж). Эти показатели важы, но позже. Сейчас главное — архитектура и логика.
- **Задача:** Создать работающую, логичную структуру сайта и процессов.

## 2. Методология Управления
Все действия должны укладываться в цикл:
1.  **Планирование (Planning):** Что делаем? Зачем? Структура. (Папка `Plan/`)
2.  **Организация (Organization):** Подготовка ресурсов, референсов, инструментов.
3.  **Координация (Coordination):** Исполнение, написание кода, связка компонентов.
4.  **Контроль (Control):** Проверка результата (Тесты, Сверка с планом).

**ВАЖНО:** Не перескакивать тапы. Не писать код без утвержденного плана.

## 3. Структура Проекта
- **Корень (`/`):**
    - `Plan/` — Вся документация по планированию. Это папка, а не файл.

    - `web/` — Next.js проект (Сайт).
    - `agents.md` — Этот файл.
