# Ветка: Автоматизация (n8n/CRM/каналы)

Цель: чтобы лид не терялся, а коммуникации и отправка ценности работали автоматически.

Минимальный контур:

- формы/квизы/чат → CRM,
- авто-уведомления в WhatsApp/Telegram,
- авто-выдача результата (PDF/подборка),
- дашборд метрик (трафик/лиды/конверсия/каналы).
