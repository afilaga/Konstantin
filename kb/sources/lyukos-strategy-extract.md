# Выжимка из `lyukos-strategy.html`

## Почему текущий сайт “не работает”

### 1) Сайт ни к кому не обращается

- Инвестору: нет ROI/гарантий/логики сделки
- Семье: нет “как переехать” / “с чего начать”
- Премиальному: нет ценности и причин выбрать
- Эмигранту: нет объяснения процесса (визы/налоги/шаги)

### 2) Нет входа (ясности за 3 секунды)

Пользователь не понимает: куда попал, кто Константин, что он делает, зачем сайт существует.

### 3) Не проговорены боли

Риски потери денег, страх обмана, юридические ошибки, неопытность в Дубае, непонимание процесса — есть, но сайт их не артикулирует → доверие не формируется.

### 4) Визуал без концепции

Слабые фото, “минимализм ради минимализма”, нет “digital showroom”, нет стиля личности.

### 5) Хаотичная структура

Случайные блоки без логики и связей, цифры без контекста.

### 6) Нет CTA

Нет действий “получить консультацию / оценить инвестицию / подобрать объект / задать вопрос”.

### 7) Нет личности Константина

Нет точки зрения, роли/позиции, историй, принципов, кейсов → не работает как личный бренд.

## Цель

Сделать Константина “THE GUY в Дубае”: инсайдер, которому доверяют деньги и решения (не “ещё один агент”).

## Что делаем (крупные блоки)

1. Позиционирование Константина как инсайдера и собственника агентства + доказательства (кейсы, истории, цифры).
2. Контент как инструмент доверия (YouTube с контекстом, инсайды, разбор ошибок, регулярность).
3. Новый сайт на React (премиум-уровень 2026): адресность по сегментам, решение болей, соцдоказательства, CTA.
4. AI-бот (цифровая версия): 24/7, мульти-язык, квалификация, персональные подборки.
5. Экспресс-приложения под видео на субдоменах (ROI/чек-листы/гиды по районам) + интеграция с CRM.
6. Умные квизы для сегментации (инвестор/семья/эмигрант) → персональный результат + захват контакта.
7. Сезонные кампании с автоматизацией (лендинги + рассылки + соцсети) по расписанию.
8. Лидогенерация через все каналы (YouTube/Instagram/Threads/Telegram/WhatsApp) → единая CRM.
9. Индивидуальные подборки объектов (AI-портрет клиента → подборка → PDF/email/Telegram).
10. Генерация презентаций (шаблоны + данные + аналитика просмотра).
11. Качество визуала уровня мировых брендов (фото/обработка/фирстиль/видео-интро/аутро).

## Этапность (в сокращении)

- Этап 0 (3–4 недели): аудит → интерактивный прототип → согласование стратегии.
- Этап 1 (4–5 недель): новый сайт (главная) + 3 экспресс-приложения + визуал.
- Этап 2 (3–4 недели): n8n-автоматизация + мультиканальные сценарии + AI + CRM/дашборд.
- Этап 3 (4–6 недель): все страницы + новые субдомены + сезонные кампании + SEO.
- Этап 4 (постоянно): оптимизация, A/B, поддержка, масштабирование.
