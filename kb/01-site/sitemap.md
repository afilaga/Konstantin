# Карта сайта и субдомены

## Основной домен (core)

1. Главная (сегментация + CTA)
2. О Константине (история, принципы, позиция, медиа)
3. Услуги (детально, отдельные страницы под направления)
4. Кейсы (истории клиентов, цифры, фото/видео)
5. Каталог объектов (если нужен: фильтры/карта/калькуляторы)
6. Контент (YouTube + статьи/гайды с контекстом)
7. Контакты (офис/мессенджеры/заявка)

## Субдомены (экспресс-приложения / лид-магниты)

- `invest.lyukos.com` — калькулятор ROI + захват контакта.
- `family.lyukos.com` — чек-лист переезда + квиз района.
- `quiz.lyukos.com` — “подберём район за 2 минуты” (универсальный квиз).
- `marina.lyukos.com` — гид по району + объекты + CTA.
- `downtown.lyukos.com` — гид по району + объекты + CTA.
- `visa.lyukos.com` — визы/статусы + “разбор ситуации”.
- `calculator.lyukos.com` — налоги/расходы/сравнение сценариев.
- сезонные: `spring.lyukos.com`, `summer.lyukos.com`, `autumn.lyukos.com`, `newyear.lyukos.com`.

## Правило маршрутизации (YouTube → сайт)

Каждое видео ведёт на “свой” субдомен/лендинг (ценность → контакт), а не на главную.
