# Ветка: Сайт (лидогенерация + личный бренд)

## Зачем сайт

Не “визитка”, а система: сегментирует аудиторию, отвечает на боли, даёт ценность в обмен на контакт, ведёт к консультации/сделке.

## Ключевая проблема текущего сайта (из анализа)

- Нет адресности под сегменты (инвестор/семья/эмигрант/премиум).
- Нет ясности “кто/что/зачем” за первые 3 секунды.
- Нет проговорённых болей → нет доверия.
- Нет личности Константина (позиции, истории, принципов, кейсов).
- Нет CTA и понятного следующего шага.

## Принципы нового сайта

- “Константин = инсайдер Дубая”: статус, опыт, принципы, доказательства.
- “Один экран — одно действие”: CTA на каждом ключевом экране.
- “Каждый сегмент видит себя”: отдельные входы/лендинги или динамический первый экран.
- “Видео → действие”: каждое YouTube-видео ведёт на экспресс-приложение/лендинг, а не на общую главную.
